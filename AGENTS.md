# AGENTS

このリポジトリは日本語縦書き脚本向けの Windows デスクトップエディタです。

## 主要参照
- `Document.md`: 確定要件と挙動ルール
- `Plan.md`: 実装マイルストーン

## 変更時の期待
- レコード単位の編集挙動と WYSIWYG レイアウト規則を維持する。
- フォントサイズ、余白、間隔などの固定値は製品判断なく変更しない。
- Enter/Tab/Ctrl+Enter の挙動は仕様に合わせる。
- アプリケーション内に表示するテキストはすべて日本語とする。

## 安全ルール
- 依頼がない限り破壊的な git コマンドは使わない。
- 自分が行っていない予期しない変更を見つけたら、停止してユーザーに確認する。
