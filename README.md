# VerticalScenarioEditer (VSE)

日本語の**縦書き脚本**（役名＋セリフ/ト書き）を、**縦書き2段組の見たまま**編集し、そのまま**PDF出力**できる Windows 向けエディタです。

> 役名（1段目）と本文（2段目）を「レコード」として扱い、脚本の入力・整形・出力を一連の流れで行えます。

## 主な機能

- 縦書き2段組の見たままで編集（本文モード）
- 横書きでの簡易編集（簡易モード）
- タイトル / あらすじ / 登場人物紹介の編集（概要モード）
- PDF出力（本文＋概要を含めて出力）
- 整形済みテキスト出力（例：`登場人物「セリフ」`）
- 範囲指定してテキスト出力（選択モード）
- 同形式テキストの読み込み
- 登場人物ごとの背景色設定（役名辞書）
- ページ番号 / 罫線 / 改行記号の表示 ON/OFF

## 対応・前提

- OS: Windows
- 文字コード: テキスト読み込みは `UTF-8`
- 保存形式: 独自形式 `.vse`

## まずは使ってみる（基本の流れ）

1. 本文モードで入力を開始
2. `Ctrl+Enter` で次のレコード（役名＋本文）を追加
3. 役名セルを入力したら `Enter` / `Tab` で本文セルへ移動して入力
4. 必要に応じて概要モードで「あらすじ」「登場人物紹介」を編集
5. PDF出力、またはテキスト出力

## 操作の要点（最初に覚える分だけ）

### 用語
- **役名セル**：登場人物名が入る部分（1段目）
- **本文セル**：セリフ・ト書きが入る部分（2段目）
- **レコード**：役名セル＋本文セルの1セット

### 入力・改行
- `Ctrl+Enter`：次のレコードを追加
- 役名セルで `Enter` / `Tab`：本文セルへ移動
- 本文セルで `Enter`：
  - **行に文字がある場合**：本文セル内で改行
  - **行が空の場合**：次のレコードへ移動（存在しなければ新規作成）
- `Shift+Enter`：本文セル内で改行（空行でも次レコードに移動しない）

### よく使うショートカット
- `Ctrl+S`：上書き保存
- `Ctrl+Z`：もとに戻す / `Ctrl+Y`：やり直す
- `Ctrl++`：拡大 / `Ctrl+-`：縮小（`Ctrl+スクロール`でも可）

## モード

- **本文モード（デフォルト）**：縦書き2段組の通常編集。PDF出力時はこの見た目になります。ページ端到達で自動改ページ。
- **簡易モード**：横書き編集（操作体系や編集対象は本文モードと同一）。
- **選択モード**：テキスト出力範囲の選択用。選択範囲のみ出力。
- **概要モード**：タイトル / あらすじ / 登場人物紹介の編集。PDFでは本文の前に差し込まれます。
  - ※概要モードは自動改ページしません。改ページは `Ctrl+Enter`。
