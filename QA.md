# 手動テストチェックリスト（M13）

このチェックはリリース前の最小確認です。

## 1. レイアウト
- 起動直後に縦書き表示される
- A4横向きのページ枠、余白が正しく見える
- 役名セルが上、本文が下に表示される
- 罫線の表示 ON/OFF が効く
- 改行・改ページ記号の表示 ON/OFF が効く（編集対象外）

## 2. 入力とキー操作
- 役名セルで Tab を押すと本文セルへ移動する
- 本文セルで Tab を押すとタブ文字が挿入される
- Shift+Tab で役名セルへ移動する
- Enter で本文内改行
- 空行で Enter すると次レコードへ移動する
- Ctrl+Enter で次レコードを新規追加して移動する
- Ctrl+Shift+Enter で前レコードを新規追加して移動する
- Ctrl+Z / Ctrl+Y で Undo / Redo が動作する

## 3. 保存と読み込み
- .vse を保存して読み込める
- 名前をつけて保存で作業ファイルが切り替わる
- 整形済みテキストから読み込みができる

## 4. 出力
- PDF 出力で A4横向きになる
- 罫線が PDF に含まれない
- ページ番号の ON/OFF が PDF に反映される
- 整形してテキストファイル出力ができる

## 5. モード
- 概要モードに切り替えできる
- 簡易モードに切り替えできる
- 選択モードの ON/OFF が動作する

## 6. 役名辞書
- 役名色が表示される
- 役名辞書の更新が反映される
